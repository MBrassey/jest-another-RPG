## jest another RPG

Command line RPG written in Node.js following the Test Driven Development model with jest.

#### Requirements

- node.js
- npm

#### Installation

    npm i

#### Usage

    Usage: node app.js [ -h | -v | -l | -a ]

    [options]

    -h          Display this message.
    -v          Show version.
    -l          Show license info.
    -a          What is jest-another-RPG?
